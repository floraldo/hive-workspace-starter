# Workspace Lifecycle

- Root `00-context/` holds stable facts, preferences, audiences, tools, and boundaries.
- Root `01-input/` holds cross-project sources and a temporary inbox.
- Root `02-output/` holds cross-project deliverables and exports.
- Root `03-status/` holds workspace state, tasks, and settled decisions.
- Root `04-knowledge/` holds reviewed cross-project learning with provenance.
- Keep project-specific material in the project's matching numbered folder.
