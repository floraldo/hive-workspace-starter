# Projects

Each project is a bounded outcome at `projects/<safe-lowercase-hyphen-slug>/` and repeats
the workspace lifecycle:

```text
00-context/
01-input/
02-output/
03-status/
04-knowledge/
```

Copy `_template/` through `/new-project` or `$new-project`. The included
`kinsai-workshop/` is a fictional teaching example, not client data.
